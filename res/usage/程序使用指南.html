字典

创建字典

  • 点击菜单，字典 -> 创建字典。 弹出对话框， 选择创建的字典位置及名称，即可创建字典。
  • 创建字典之后， 界面跳到字典界面编辑模式， 可以编辑字典首页内容， 编辑完成 点击保存， 或者快捷键 ctrl + s
  • 如果不在字典首页， 可以通过菜单， 字典， 点选相应的字典打开字典首页。

关闭字典

  • 点击菜单，字典， 关闭字典，即可关闭现在字典。

打开字典

  • 点击菜单， 字典 -> 打开字典，选择字典，打开即可。

词条

创建词条

  • 打开字典后， 点击条目， 添加条目，跳转到添加条目页面， 输入条目名称， 和条目内容， 点击保存，或者 ctrl+s 保存。
  • 也可以点击左下角的 "+" 创建条目

删除词条

  • 点击 菜单 条目 -> 删除条目 或者点击左下角的 "-" 也可以删除该条目。

查找词条

  • 在搜索框内输入要查找的词条名称，即可，
  • 可以点击左下角的选择框， 选择排序规则。

其他

查看模式

  • 点击右下角下角的眼睛， 即可进入查看模式

              

放大与缩小

  • 点击右小角的 放大镜，缩小镜 可以放大缩小， 也可以用，快捷键 ctr+- 和 ctr+shift++

                 

  • 若想回到当前大小 ctrl + 0

dict文件。

  • dict 文件是 sqlite的 db 文件。
  • 这个文件包含所有的词条。

图片编辑

  • 图片可以直接导入，但不能再markdown下面编辑。

链接

  • 内部链接到其他的词条默认以 entry:// 开头。

             

  • 如果链接到网络， 以http:// 开头。
  • 不建议链接到本地文件
