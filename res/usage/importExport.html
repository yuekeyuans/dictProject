导出与导入

程序支持将字典导出为 pdb样式的html ， 用于在程序之间传递数据的excel 文件，可预览的html 样式以及导出为字典样式的dictHtml 格式文件，并支持将字典项导入到字典中来。

导出为 pdb样式的html文件。

  • 打开要导出的字典， 点击菜单 导出 -> pdb源文件。 选择要导出的位置。文件会自动导出到所选择的文件夹位置。
  • 打开pdb 转换程序， 选择index.html 进行导出。
  • 注意，使用isilox 导出时，编码选择 GB2312

导出为 html

  • 打开要导出的字典，点击菜单 导出 -> html 网页。 选择要导出的位置。文件会自动导出到所选择的位置。
  • html 文件可以用在没有字典程序的情况下使用，直接打开 index.html,在浏览器中查看。

导出为 excel

  • 打开要导出的字典， 点击菜单 导出 -> excel文件。 选择要导出的位置。文件会自动导出。
  • excel 文件为在程序之间来回传递数据的媒介。比如要将一个字典的词条导入到，另外一个字典，就可以先导出到excel 文件，然后再在另一个字典中导入excel。这样数据就从一个字典导入到另外一个字典了。

导出到 dictHtml 字典。

  • 打开要导出的字典， 点击菜单 导出 -> dictHtml字典。 选择要导出的位置。文件会自动导出。
  • 打开 MdxBuilder.exe (mdc 转换文件)。
    • 来源选择导出的 mHtml 文件
    • 目标选择 要到处的位置和名称
    • 格式不选择
    • 原先的格式选择 MDict(Html)
    • 编码选择 GBK 默认编码
    • 名填写要导出的名称
    • 点击开始，即可导出文件

  • 关于MdxBuilder.exe 中的描述
    • 描述可以不填写，如果要填写，请按照一下步骤：
    • 打开要导出的字典， 选择菜单 字典，点击要导出的字典，主界面会显示字典的说明。
    • 点击编辑菜单上的 类似 5 的按钮， 菜单中央会出现文本的网页编码。
    • 复制网页编码到MdxBuilder.exe 的一栏即可。

导入为 excel

  • 点击菜单 导入 -> excel 文件， 选择文件，即可导入。
  • excel 文件是用于字典中间的文件传递。
